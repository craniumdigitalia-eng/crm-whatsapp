# Story 5.12 — Etiquetas (tags) nos leads
**Pedido do usuário (2026-06-25).** Permitir criar e aplicar **etiquetas coloridas** nos leads do kanban.

## Escopo
- **Schema (crm-data):** tabela `tags` (id, nome, cor) + `lead_tags` (lead_id, tag_id) — many-to-many. RLS coerente.
- **API (crm-backend):** CRUD de tags + atribuir/remover tag de um lead.
- **UI (crm-frontend):** chips de etiqueta no card do kanban e no drawer; um gerenciador de etiquetas (criar/editar/cor); filtrar o kanban por etiqueta.
## Aceite (rascunho)
- AC1: criar etiqueta com nome + cor. AC2: aplicar/remover N etiquetas por lead. AC3: chips visíveis no card e drawer. AC4: filtro por etiqueta no board. AC5: persiste no Supabase.
